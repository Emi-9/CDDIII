- **analisis**:

        - `densidad` (estimacion de densidades por kernel)

        - `azar` (genera un conjunto muestral de datos siguiendo una distribucion)

- **modelos**:
    - **regresion_lineal_simple**:

    - **regresion_lineal_multiple**:
        - `completar` ()
